- Powershell works with unix commands too i.e. linux commands also works in powershell.

Unix | cmd | Powershell
--- | --- | ---
cd home | cd c:\ | set-location c:\
ls | dir | get-childitem
clear | cls | clear-host
mkdir dirname | mkdir dirname | md dirname
man | help | help
